## 图片自动嵌入功能实现

用户要求在笔记中添加论文图片（架构图、关键图）。

### 实现的改进

1. **描述性文件名**: 图片命名从 `Pasted image {ts}_{n}.png` 改为 `{paper_name}_fig{N}.png`（如 `TiMi_fig3.png`）
2. **自动 caption**: 从 `figure_notes` 提取图片说明，以 blockquote 格式（`> Caption`）插入图片前
3. **Heading-based 位置匹配**: 当架构文本无 figure reference 时，基于 caption 关键词与 ### heading 的语义匹配来确定插入位置
4. **图片分类**: 根据 figure_notes caption 关键词将图片分为 architecture/experiment 两组

### 修复的 Bug

- **Bug 4**: `_cleanup_figure_table_refs` 误删 `![[TiMi_fig3.png]]` 中的数字——`Fig\.?` 匹配了 `fig3`，将 `3` 当作 figure 编号删除，文件名被破坏为 `TiMi_ .png`
  - 修复：在 `_cleanup_figure_table_refs` 中添加 Obsidian embed 保护机制（临时替换 placeholder → 清理 → 还原）
- **Bug 5**: `_cleanup_architecture` 的 `^>\s?` regex 将所有 blockquote 行转为普通文本，包括图片 caption
  - 修复：在 `_cleanup_architecture` 中添加 embed+caption 保护机制
